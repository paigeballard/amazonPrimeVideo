Amazon Prime Video Clone
This project is a simple clone of the Amazon Prime Video interface, built using HTML, CSS, and JavaScript. It fetches movie data from the TMDB API to dynamically display movie information, including a hero section and a carousel of related movies.

Features -
Hero Section: Displays a featured movie with its title, release date, and overview.
Search Functionality: Allows users to search for movies by title.
Related Movies Carousel: Shows a list of movies related to the featured movie.
Responsive Design: Adapts to different screen sizes for a better user experience.

Technologies Used -
Frontend
HTML: Structure of the web page.
CSS: Styling and layout, including responsive design.
Custom styles in style.css.
Font Awesome for icons.
Google Fonts for typography.
JavaScript: Dynamic functionality and API integration.
Fetching movie data from the TMDB API.
Updating the DOM with movie information.
API
TMDB API: Used to fetch movie data, including search results and related movies.
